To use the program, follow these steps\+:
\begin{DoxyItemize}
\item download the source files using \texorpdfstring{$<$}{<}git clone \href{https://github.com/roxerj/uni2/tree/v2.0}{\texttt{ https\+://github.\+com/roxerj/uni2/tree/v2.\+0}}\texorpdfstring{$>$}{>} or download the zip file
\item open terminal in the directory where the files are located
\item type make and press enter (make sure you have make installed)
\item follow the steps in the program, you can generate a file that contains a chosen number of students or use an existing file
\item the program will generate two files\+: vargsai.\+txt and malaciai.\+txt, containing the students that failed and passed the course, respectively
